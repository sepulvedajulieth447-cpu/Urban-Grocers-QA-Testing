QA Testing Project - Urban Grocers 🛒
Este proyecto contiene la documentación y ejecución de pruebas de API para el servicio de entregas "Order and Go".
🚀 Alcance del Proyecto
Diseño de Pruebas: Creación de matriz de casos de prueba (Caja Negra) utilizando técnicas de valores límite.
Ejecución Técnica: Validación de endpoints (POST/GET) con Postman.
Gestión de Defectos: Documentación detallada y seguimiento de errores en Jira.
📁 Documentación Incluida
Casos de Prueba:Archivo PDF con escenarios de prueba, datos de entrada y resultados.
Reporte de Bugs:Ejemplo técnico de reporte de fallo (Bug TRIP-27) con pasos de reproducción.
Evidencias Postman: Capturas de las peticiones JSON y respuestas del servidor.
Herramientas utilizadas: Postman, Jira, Google Sheets.
